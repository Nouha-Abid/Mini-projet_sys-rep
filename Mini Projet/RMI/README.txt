                           # Java RMI #

Ce projet implémente un système de gestion de liste de tâches en utilisant Java RMI.Ajout, Supression et Affichage


Etapes:

   1. Créer un nouveau projet 
   2. Copiez les fichiers source (Server.java, RmiTraitImpl.java, RmiTrait.java) à partir de "Mini Projet\RMI\RMIServer\src"   et   (Client.java) à partir de "Mini Projet\RMI\RMIClient\src" dans le dossier de votre projet.
   3. S'assurer que le port 1125 n'est pas occupé et Lancer le serveur. 
   4. Lancer le client
