                    #  Java Socket #

Ce projet implémente un service de chat en utilisant les sockets JAVA.


Etapes:

   1. Créer un nouveau projet 
   2. Copiez les fichiers source (Chat.java, Client.java) à partir de "Mini Projet\Socket\Socket\src" dans le dossier de votre projet.
   3. S'assurer que le port 9090 n'est pas occupé et lancer le serveur
   4. Lancer le Client et entrer le nom d'utilisateur.répéter cette étape.Presser Entrer dans les autres clients pour voir les messages envoyes
